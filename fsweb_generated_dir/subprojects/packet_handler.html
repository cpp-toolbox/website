packet_handler

to use the packet handler you have to create a subproject called packet_types that defines a struct as follows:

packet_type.hpp cpp enum class PacketType : uint8_t { UNIQUE_CLIENT_ID, GAME_UPDATE_POSITIONS, GAME_UPDATE_ONE_TIME_EVENTS, USER_MESSAGES, }; when you run sbpt it will correctly include it.

source code

git submodule add --recurse-submodules git@github.com:cpp-toolbox/packet_handler.git

subproject dependencies

  • packet_types
  • client_networking
  • server_networking
