When working with the toolbox a common thing you'll want to do is get graphics up and going on a project, to set that up, go to your projects root directory and do the following 

    
mkdir -p assets/
git submodule add git@github.com:cpp-toolbox/shaders.git assets/shaders 

mkdir -p src/graphics/
git submodule add git@github.com:cpp-toolbox/batcher.git src/graphics/batcher 
git submodule add git@github.com:cpp-toolbox/shader_cache.git src/graphics/shader_cache

python scripts/sbpt/main.py src
git submodule update --init --recursive
  • Next setup the `shader_cache` 
        
python scripts/setup/graphics_systems.py --setup shader-cache
  • If you plan on using the batcher then run 
        
python scripts/setup/graphics_systems.py --setup batcher

Next be sure to grab glfw and glad
