cpp toolbox provides some good code to handle input from the keyboard and mouse so you don't have to spend writing that code and just start making games in c++: 

    
cd src/input
git submodule add git@github.com:cpp-toolbox/input_state.git
git submodule add git@github.com:cpp-toolbox/glfw_lambda_callback_manager.git
cd -

python scripts/sbpt/main.py src
git submodule update --init --recursive
