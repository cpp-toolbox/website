When working with the toolbox a common thing you'll want to do is get graphics up and going on a project, to set that up, go to your projects root directory and do the following

input


mkdir -p src/input && cd input
git submodule add git@github.com:cpp-toolbox/window.git
git submodule add git@github.com:cpp-toolbox/glfw_lambda_callback_manager.git
git submodule add git@github.com:cpp-toolbox/input_state.git


    
cd assets
git submodule add git@github.com:cpp-toolbox/shaders.git
cd -

cd src/graphics
git submodule add git@github.com:cpp-toolbox/batcher.git
git submodule add git@github.com:cpp-toolbox/shader_cache.git
git submodule add git@github.com:cpp-toolbox/window.git
cd -


python scripts/sbpt/main.py src
git submodule update --init --recursive
  • Next setup the `shader_cache` 
        
python scripts/setup/graphics_systems.py --setup shader-cache
  • If you plan on using the batcher then run 
        
python scripts/setup/graphics_systems.py --setup batcher

looking and moving


mkdir -p src/utility && cd src/utility
git submodule add git@github.com:cpp-toolbox/fps_camera.git

geometry


mkdir -p src/graphics && cd src/graphics
git submodule add git@github.com:cpp-toolbox/vertex_geometry.git
