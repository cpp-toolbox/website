Move and Edit mode

This mode allows you move around efficiently, use it to make selections or move to where you need to be and then use insert mode to type what you need. By pressing escape you can always get back to this mode, it is the base mode.

  • j: move down
  • k: move up
  • l: move left
  • ;: move right
  • o: open a new line below the current
  • p: paste the last deleted thing
  • d: delete the current line
  • u: undo the last change
  • r: redo the last change

Command Mode (press c)

In this mode you can run commands related the file.

  • f: save the file
  • q: quit the program

Insert mode (press i)

In this mode you can type out regular text

Visual Select mode (press v)

while in this mode the regular movements j, k, l, ; apply

  • x: delete the active selection
  • y: copy the active selection to clipboard
